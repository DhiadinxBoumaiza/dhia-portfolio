Dhia Eddine Boumaiza — Responsive Portfolio

Open index.html in your browser.

Folders:
- assets/logos/: put technology logos here. Use the filenames listed in assets/logos/README.txt.
- assets/projects/: project screenshots are not used in this version. The project cards use clean placeholders instead.
- assets/dhia.jpg: profile photo. You can replace it with another photo; the website crops it automatically.

Recommended image sizes:
- Technology logos: 512 x 512 px, transparent PNG.
- Profile photo: square photo, at least 1000 x 1000 px.

Main files:
- index.html
- css/styles.css
- js/script.js
